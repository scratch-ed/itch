# square-test
Opencv square tester for scratch

First install the dependencies. 

```code
npm install 
```

Then run the code:
```code
node test/square-test.js 'filename.sb3'
```

The expected output is:

```code
TAP version 13
# Subtest: isSquare
    ok 1 - Goed gedaan, je hebt een vierkant getekend
    1..1
ok 1 - isSquare # time=5035.415ms
```
